## 概要

MiPA は[Discord.py](https://github.com/Rapptz/discord.py)
ライクな書き方ができるように作っている MisskeyApi wrapper です

## 注意

- このプロジェクトは開発中です。仕様が定まっていないため、破壊的変更が多いです。
- `master` ブランチで使用しているmipacは`GitHub`にあるmipacの`master`ブランチの物です。
